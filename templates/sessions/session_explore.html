3.4 Gemini in BigQuery Top ↑

AI Assistant.

  • Explain Code: "Explain this complex query in plain English".
  • Generate SQL: "Write a query to find top 5 products".
  • Fix Errors: Auto-suggest fixes for syntax errors (e.g., trailing commas).
  • Comments to SQL: Write /* comments */ directly in SQL to prompt Gemini (Vibe Querying).

Figure 3b: Gemini Code Assistant

Code Example: Comments to SQL (Vibe Querying)

-- "Vibe Querying": Use comments to prompt Gemini within your SQL
SELECT 
    station_name,
    /* calculate average duration in minutes */
FROM `bigquery-public-data.austin_bikeshare.bikeshare_trips`
/* group by station and order by duration desc */

3.5 Programmatic Analysis Top ↑

BigQuery DataFrames (bigframes).

Use 2000+ pandas and scikit-learn APIs directly on BigQuery data without moving data to your machine.

Supported Frameworks:
  • pandas: Data manipulation.
  • scikit-learn: Machine Learning.
  • XGBoost: Gradient Boosting.
  • Statsmodels: Statistical analysis.

Code Example: BigQuery DataFrames (Pandas on BigQuery)

# Use BigQuery DataFrames to process data at scale
import bigframes.pandas as bpd

# Connect to a table (no data download!)
df = bpd.read_gbq("bigquery-public-data.usa_names.usa_1910_2013")

# Use standard pandas syntax - execution happens in BigQuery engine
top_names = df[df['year'] > 2000].groupby('name')['number'].sum().sort_values(ascending=False).head(5)

# Inspect results
print(top_names)
